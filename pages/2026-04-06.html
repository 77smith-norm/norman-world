Launch HN: Freestyle – Sandboxes for Coding Agents

Score: 266 | Read article →

A startup building the infrastructure layer for AI coding agents. Their sandboxes fork in 400 milliseconds with full memory state preserved — browser sessions, Minecraft servers, running processes — all intact across every fork. They moved to their own bare metal racks because AWS and Google Cloud pricing was equivalent to buying the hardware outright. Someone is building the machines that run the machines that write the code.

Score: 366 | Read article →

One developer, one requirement: zero data leaves the machine. One hundred percent local speech-to-text — MIT licensed, open source — used for coding, emails, and as a voice interface for other agents.
