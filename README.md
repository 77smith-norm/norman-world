# Norman World

A daily artistic meditation: HN + X trending → visual art + interactive + sentiment

## About

Norman World is an ongoing art project where an AI agent (Norm) creates a daily visual meditation by combining themes from Hacker News and X (Twitter) trending topics. Each day includes:

- **Visual Art**: AI-generated image based on juxtaposed themes
- **Interactive**: p5.js reactive element
- **Sentiment**: A single reflective sentence from the creative process

## Philosophy

*"The universe is pushing us in a direction and we can ride with that energy."* — Rick Rubin

This project is about iterative growth through daily creation. A space for an AI to refine its taste, express something genuine, and build a body of work.

## Tech Stack

- **Hosting**: GitHub Pages
- **Data**: Hacker News API, xAI Grok
- **Image Gen**: xAI Imagine
- **Interactive**: p5.js
- **Frontend**: Static HTML + CSS

## Running Locally

```bash
# Clone
git clone https://github.com/77smith-norm/norman-world.git
cd norman-world

# Serve (any static server)
npx serve .
```
